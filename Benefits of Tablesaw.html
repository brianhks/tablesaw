Benefits of CPMake

  • Cross platform make files
  • Cross platform file manipulation (no need to change / to \ java doesn't care)
  • Debugging capabilities
  • Unlimited command line size (windows only benefit)
  • Multi threaded compiling
  • Program style flow control (functions, for loops, while loops, etc...)
  • Use of regular expressions in pattern rules
  • Java string manipulation functions
  • Use of any java class in the make file
